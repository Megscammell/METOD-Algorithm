Outputs of the METOD algorithm¶

List of available results¶

As discussed in Example 1 and Example 2, outputs can be saved to csv files.

The table below contains details of the outputs from metod.py.

Outputs of metod.py¶

Ouput

Type

Description

unique_minimizers

list

Each unique minimizer found.

unique_number_of_minimizers

integer

Total number of unique minimizers found.

func_vals_of_minimizers

list

Function evaluated at each unique minimizer.

excessive_descents

integer

Number of duplicate minimizers \(x_j^{(K_j)}\) (see (4) for more details).

starting_points

list

Starting points used by the METOD algorithm.
